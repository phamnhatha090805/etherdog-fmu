# etherdog-fmu
This project is to simulate difference EtherCAT device with FMU modeling. The idea is using KickCAT project to emulate virtual EtherCAT device and FMU model serve as virtual systems. The PDO/SDO/ of the KickCAT will be connected to the variables of the FMU model, this way the behaviour of the virtual EtherCAT device can be simulated and visible.
